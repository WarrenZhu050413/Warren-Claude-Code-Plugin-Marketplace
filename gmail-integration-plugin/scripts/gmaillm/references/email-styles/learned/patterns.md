## Common Patterns

### Apology Emails
- Start with: "Sorry for..."
- Be direct about the issue
- Offer solution or timeline

### Follow-up Emails
- Reference previous exchange
- Maintain same tone as original thread
- Keep it brief

### Update Emails
- Use bullet points for multiple items
- Lead with most important information
- End with next steps if applicable
